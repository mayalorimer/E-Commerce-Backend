# E-Commerce-Backend

## Description
This application provides the back end for an e-commerce site. Various API routes are used to view data from the database as well as to add to it, modify existing data, and delete data. 

## Technologies Used
- Javascript
- Node.js
- Express.js
- Sequelize
- MySQL
